A repo where I store all the code I did on my laptop.
It contains code from some NeetCode problems and how I solved problems from Striver DSA course

> Step 1: Create a `python` file to solve a particular problem.

>Step 2: Import `Engine` `class` from `engine.py`

>Step 3: Create `object` of `Engine` `class` and pass the test cases for that problem

Use `engineinit` to insert default template in python files.

>Step 4: Solve the problem.

>Step 5: Run the code with the `v8` `method` of the `Engine` `class`.

>Optional 1: If you have multiple solutions, compare them with `compare` `method` of the `Engine` `class`

>Optional 2: If you want to guess the complexity, use the `scale` `method` of the `Engine` `class` which tries to guess the complexity of your code.
